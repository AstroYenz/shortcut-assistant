Shortcut Assistant: Updated


Shortcut Assistant has been updated to version 2.2.0, a release that includes 2 new features:


  1. Search for Shortcut stories using the Chrome Omnibox: Simply type "sc" followed by a tab and your search query to get started.
  2. Story Analysis responses from OpenAI are now streamed back so responses feel snappier (OpenAI token in settings required).
